# Break Tetris v${{ github.ref_name }} - The Hybrid Challenge Continues!

## 🚀 How to Play
Download the correct file for your operating system from the Assets section below. Follow the instructions for your system.

### 🪟 Windows
1. Unzip `break-tetris-win.zip`.
2. Run `break-tetris.exe` inside the new folder.
3. If a Windows SmartScreen warning appears, click "More info" -> "Run anyway".

### 🍎 macOS
1. Unzip `break-tetris-mac.zip`.
2. Run the `break-tetris` application.
3. If you see a security warning, go to "System Settings" > "Privacy & Security" and click "Open Anyway".

### 🐧 Linux (AppImage)
1. Download the file (e.g., `break-tetris-x86_64.AppImage`).
2. Make it executable using the terminal:
   ```bash
   chmod +x break-tetris-x86_64.AppImage
   ```
3. Run the game:
   ```bash
   ./break-tetris-x86_64.AppImage
   ```

## 🎮 Controls
| Key | Action |
| :--- | :--- |
| **Arrow Keys / H, L** | Move Piece / Paddle |
| **Up Arrow / K** | Rotate (Tetris) |
| **Down Arrow / J** | Soft Drop (Tetris) |
| **Space** | Hard Drop (Tetris) / Slam (Breakout) |
| **F** | Toggle Fullscreen |
| **P** | Pause Game |
| **M** | Mute/Unmute Sound |
| **G** | Toggle Ghost Piece (Tetris) |
| **C** | Switch Mode (Debug) |
